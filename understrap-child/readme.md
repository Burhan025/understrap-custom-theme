# Understrap Child Theme (Portfolio Version)

This repository contains a **sanitized, portfolio-safe** Understrap child theme that demonstrates:
- Custom theme structure & template overrides
- ACF blocks + JSON sync patterns
- WooCommerce template customization patterns
- Build scaffolding (`src/sass`, `src/js`)

> Private documentation links, proprietary identifiers, and secrets were removed or replaced with placeholders.

## Requirements
- WordPress
- Understrap parent theme (`understrap`)

## Secrets
- Google Maps: provide `GOOGLE_MAPS_API_KEY` via environment or a secure WP option.

## Notes
This is a public/demo edition intended for code review and portfolio purposes.
